How To Use

Guide for staff & management

Getting Started

  • Navigation: Use the top menu to switch between Overview, Monthly Data, Group Totals, and this guide.
  • Monthly Data is where all daily entries are made. Select a month tab, then a club tab.
  • Data is saved automatically in your browser's local storage — no internet required after loading.
  • Overview and Group Totals update automatically as you enter data.

Entering Daily Data

  • Date: Each row represents one working day. Use the date picker to set the date.
  • Hours Worked: Enter total hours worked at that club on that day.
  • Retention Calls: Number of retention calls made.
  • Member Calls: Number of general member calls made.
  • Arrears Calls: Number of arrears follow-up calls made.
  • Arrears Collected ($): Dollar amount of arrears collected that day.
  • Notes: Any free-text notes for that day.

Tips & Best Practices

  • Click + Add Row to add more entries under a club for a given month.
  • Click the Save button after entering data to ensure it is stored.
  • The Row Total column sums all calls for that row automatically.
  • Charts on the Overview and Group Totals pages update as soon as data is saved.
  • To clear all data, open browser developer tools → Application → Local Storage and clear the site data.
  • For best results, enter data daily rather than in bulk at the end of the month.
